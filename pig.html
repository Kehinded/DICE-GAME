dice game

game rules and regulations
  • Clicking on button ROLL-DICE or pressing key(D) will add each dice values as current score accumulatively for the current active player.
  • Clicking on button HOLD or pressing key(H) will add current score to each player points and change active player to next player.
  • Clicking on button RESET will automatically reset round score, each players points back to zero(0) and reduce life(s) left by 1.
  • Getting a double 6 dice value automatically add a life for the current active player and number(s) of life(s) left is equivalent to numbers of chance to reset game.
  • Clicking on button NEW-GAME will automatically set the game back to the home page for a new start.
  • Getting dice value one (1) in any dice(s) value after a roll render current score invalid and change active player to the next player.
  • First player to score set winning value or 100 points and above if winning value is not set, wins the game.
  • To activate and start game, input player names and click on the button play game below, GOODLUCK .
